# Snake Game (Turbo C Version) 
A simple snake game written in C for Turbo C compiler.
## How to Play
- `W` → Move Up
- `S` → Move Down
- `A` → Move Left
- `D` → Move Right
- `X` → Exit
## Compile &amp; Run 
1. Open Turbo C / Turbo C++ IDE. 
2. Load `SNAKE_TURBO.C`.
3. Compile (`Alt + F9`) and Run (`Ctrl + F9`).
Enjoy the game! 🎮🐍
